  # dedicated REMEDY


# 脚本参数说明：
  -h ———— 帮助文档
  
  -u ———— 输入remedy 用户名
  
  -p ———— 输入remedy 密码
  
  -d ———— driver 路径

# 注意事项：
  由于 remedy 网站不可重复登录，所以脚本退出时务必先将 remedy 账号注销登录，以免下次使用时出现登录异常。

  使用时请确保参数中输入账号信息以及驱动路径正确，以免出现登录错误或者驱动异常。
